Kasetsart University Chalermphrakiat Sakon Nakhon Province Campus Map Application

Version: {{version}}

Application is not yet complete. Some features and content might be missing, bugs might be present. Please report all the bugs via email.

The purpose of this application is to aid university students navigate in and around the university campus, provide them useful information about the surroundings, university and university staff.

The application works offline (no internet connection needed).

GPS module is required for some of the features and accurate positioning. Turn on and allow this application to use GPS. Note that enabling GPS (High Accuracy) consumes your battery at increased rate.
